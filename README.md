# الأفق ميديا - صفحة هبوط

صفحة هبوط احترافية لشركة الأفق ميديا للدعاية والإعلان.

## المحتوى

- **index.html** - الملف الرئيسي الذي يحتوي على كل الموقع

## الميزات

- ✅ تصميم احترافي وعصري
- ✅ متجاوب على جميع الأجهزة
- ✅ سرعة تحميل عالية
- ✅ محتوى عربي كامل
- ✅ نموذج اتصال فعّال

## الأقسام

1. **رأس الصفحة** - قائمة التنقل والشعار
2. **قسم البطل** - العنوان الرئيسي والدعوة للعمل
3. **عن الشركة** - معلومات عن الأفق ميديا
4. **الخدمات** - 6 خدمات رئيسية
5. **آراء العملاء** - شهادات العملاء
6. **التواصل** - نموذج الاتصال ومعلومات التواصل
7. **التذييل** - روابط سريعة ومعلومات

## الألوان

- **الأزرق الداكن**: #001f3f (اللون الأساسي)
- **الذهبي**: #FFD700 (لون التركيز)

## الخطوط

- **Cairo** - للعناوين
- **Tajawal** - للنصوص الأساسية

## الاستخدام

ما عليك سوى فتح ملف `index.html` في المتصفح!

## النشر على GitHub Pages

1. ادفع الملفات إلى GitHub
2. اذهب إلى Settings > Pages
3. اختر "Deploy from a branch" و "main" و "/ (root)"
4. سيكون الموقع متاحاً على: `https://username.github.io/ofoq-media-landing-page/`
